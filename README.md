<h2 align="center">Приложение "Список дел"</h2>
<div>Реализован сайт для хранения и редактирования списка дел. На сайте можно создать задачу, отредактировать её и удалить. Можно просмотреть список всех задач, список текущих задач (время выполнения которых запланировано на ±12 часов), список выполненных задач.</div>
